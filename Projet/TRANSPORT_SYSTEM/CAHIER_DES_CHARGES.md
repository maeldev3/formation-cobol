# CAHIER DES CHARGES - SYSTEME DE GESTION DE TRANSPORT

## OBJECTIF
Développer un système complet de gestion de transport en COBOL.

## FONCTIONNALITES
1. Gestion des bus (CRUD)
2. Gestion des conducteurs (CRUD)
3. Gestion des trajets (CRUD)
4. Rapports d'occupation et CA
5. Export CSV et sauvegarde

## TECHNOLOGIES
- Langage: COBOL (GnuCOBOL)
- Base de données: SQLite
- Makefile pour compilation

## LIVRABLES
- Code source COBOL
- Scripts SQL
- Makefile
- Documentation
